## Technology
- PHP 8.0 or higher
- composer
- docker
- node.js

## Installation
```
composer create-project drago-ex/project
```

## npm
```
npm i
```

## docker build
```
docker-compose build
```

## docker dev up
```
docker-compose -f docker-compose.yml -f docker-compose.dev.yml up
```
